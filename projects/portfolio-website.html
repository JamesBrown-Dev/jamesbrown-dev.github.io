Challenges & Solutions

Challenge: Responsive Design Across All Devices

Making the site look great on everything from small phones to large desktop monitors was tricky. I had to learn about media queries, flexible layouts, and relative units.

Solution: I used CSS Grid and Flexbox for flexible layouts, combined with media queries for breakpoints. Using clamp() for font sizes helped create smooth typography scaling across screen sizes.

Challenge: Smooth Animations Without Janky Performance

I wanted smooth animations but noticed some lag on lower-end devices, especially with scroll-triggered effects.

Solution: I used the Intersection Observer API instead of scroll events for better performance. I also stuck to animating transform and opacity properties, which are GPU-accelerated.

Challenge: Version Control with Git

As a beginner, understanding Git workflow and avoiding mistakes like committing to the wrong branch was confusing at first.

Solution: I spent time learning Git fundamentals through practice. I set up a workflow of: edit ‚Üí save ‚Üí stage ‚Üí commit ‚Üí push. VS Code's Git integration made this much easier to visualize.
