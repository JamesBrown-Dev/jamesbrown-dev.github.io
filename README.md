# James Brown - Portfolio Website

A modern, responsive portfolio website built from scratch to showcase full-stack development skills.

 portfolio live at: **https://jamesbrown-dev.github.io**

## üöÄ Quick Start

### Customizing Your Portfolio

Before uploading to GitHub, personalize these sections:

#### 1. **About Section** (index.html, lines ~65-95)
- Update the bio text to reflect your own journey and interests
- Modify the stat cards if desired

#### 2. **Projects Section** (index.html, lines ~97-155)
- Replace placeholder projects with your actual projects
- Update project descriptions, technologies used, and links
- Add GitHub repository links
- Add live demo links when available

#### 3. **Skills Section** (index.html, lines ~157-230)
- Adjust skill percentages based on your comfort level
- Add or remove skills as needed
- The percentages are set with `style="--progress: 75%"` - change the number to your level

#### 4. **Contact Information** (index.html, lines ~232-280)
- Update email address (replace `your.email@example.com`)
- Add your LinkedIn profile URL
- Your GitHub link is already set to `https://github.com/jamesbrown-dev`

#### 5. **Optional Customizations**
- Change color scheme in `styles.css` (lines 3-15, CSS variables)
- Modify fonts (currently using Archivo and JetBrains Mono)
- Adjust spacing and sizes
